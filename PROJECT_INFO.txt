This is a complete Android mobile application project.

PROJECT TYPE: Android Mobile App (Gradle)
LANGUAGE: Kotlin
MIN SDK: Android 4.4.2 (API 19)
TARGET SDK: Android 15 (API 35)

===============================================
           HOW TO COMPILE & RUN
===============================================

This project MUST be compiled using Android Studio
on your local computer. It cannot run on Replit
because it requires:

1. Android SDK
2. Android Build Tools
3. Android Emulator or Physical Device

STEPS TO COMPILE:
1. Download this entire TurboLiteOptimizer folder
2. Open Android Studio
3. Select File → Open
4. Navigate to TurboLiteOptimizer folder
5. Click OK to import
6. Wait for Gradle sync
7. Click Run button

See README.md for detailed instructions.
